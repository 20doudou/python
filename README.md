# 🐍 Python 学习仓库

> 从零开始，系统学习 Python 编程

## 📖 学习路线

### 第一阶段：Python 基础（1-2 周）
- [ ] 变量与数据类型（int, float, str, bool, list, dict, tuple, set）
- [ ] 运算符与表达式
- [ ] 输入与输出（input / print）
- [ ] 条件语句（if / elif / else）
- [ ] 循环（for / while）
- [ ] 函数定义与调用
- [ ] 字符串操作

### 第二阶段：进阶语法（2-3 周）
- [ ] 列表推导式与生成器
- [ ] 装饰器（Decorator）
- [ ] 面向对象编程（类、继承、多态）
- [ ] 异常处理（try / except / finally）
- [ ] 文件读写（open / with）
- [ ] 模块与包（import / pip）

### 第三阶段：实用技能（3-4 周）
- [ ] **爬虫基础** — requests + BeautifulSoup
- [ ] **数据处理** — pandas + numpy 入门
- [ ] **数据可视化** — matplotlib / seaborn
- [ ] **自动化脚本** — os / shutil / schedule
- [ ] **正则表达式** — re 模块

### 第四阶段：项目实战
- [ ] 项目一：天气查询小工具
- [ ] 项目二：网页爬虫（抓取新闻/图片）
- [ ] 项目三：数据分析报告（Excel/CSV 数据分析）
- [ ] 项目四：简单的 Web 应用（Flask）
- [ ] 项目五：自动化办公脚本

## 💡 学习建议

1. **每天写代码**，哪怕只有 30 分钟
2. **遇到问题先搜索**，Stack Overflow 是你的好朋友
3. **多做项目**，看教程不如动手写
4. **记录笔记**，把踩过的坑都记下来
5. **参与开源**，GitHub 上有很多适合新手的项目
